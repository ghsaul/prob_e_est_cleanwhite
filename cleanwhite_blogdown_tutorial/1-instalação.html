1 Instalação

  • Tenha o R instalado

  • Tenha o RStudio instalado

  • Rode o comando install.packages("blogdown") no console do R para instalar o pacote Blogdown

  • Rode o comando blogdown::install_hugo() no console do R para instalar o Hugo

  • Rode o comando blogdown::update_hugo() no console do R para atualizar o Hugo, só para garantir que ele estará em sua versão mais recente
